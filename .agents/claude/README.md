# Claude Code files under `.agents/claude`

These files are symlinked into `~/.claude/` by `script/setup-claude-code.sh`. **`skills/`** for Claude is **not** here: it uses **`../skills/`** (shared with the skills CLI).

Imported layout originally followed [freekmurze/dotfiles](https://github.com/freekmurze/dotfiles) `config/claude/`; this repo stores them under **`.agents/claude/`** so all tooling uses **`.agents`** as one tree.
