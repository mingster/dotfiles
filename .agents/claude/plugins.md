# Claude Code Plugins

These plugins are installed via the Claude Code UI or `/plugin install <name>`.
They are not scripted — install manually on a new machine.

## User-scope (install once per machine)

| Plugin | Source | Purpose |
|--------|--------|---------|
| `context7` | claude-plugins-official | Up-to-date library and framework docs |
| `Notion` | claude-plugins-official | Notion workspace integration |
